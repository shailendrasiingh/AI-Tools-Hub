Frequently Asked Questions

Find answers to common questions about our platform

How often is the tool database updated?

We update our database weekly with new tools and regularly review existing entries to ensure they're still available and relevant.

How can I suggest a new tool to be added?

You can suggest new tools through our contact form. Please include the tool name, URL, and a brief description.

Are the tools free to use?

Many tools offer free tiers or versions, but some may require payment for full functionality. We always indicate pricing information when available.

How are tools categorized?

Tools are categorized based on their primary function and use case. We use a multi-level categorization system to make finding the right tool easier.

Do you have an API?

We're currently working on an API that will allow developers to access our tool database. Stay tuned for updates!

How can I report an issue with a tool listing?

If you find incorrect information or a broken link, please use our contact form to report the issue. We appreciate your help in maintaining accurate information.
